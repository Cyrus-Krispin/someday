Viewport Navigation: How should players move?

Game-design.md lists this as an open question. Choose the approach for the 15×15 tile viewport.

Viewport Navigation Options

Option A: Arrow Keys

↑
← ↓ →

Pros: Keyboard native, precise control

Cons: Mobile unfriendly, needs on-screen buttons

Option B: Tap to Move

Tap adjacent tile → Player moves there

Pros: Mobile native, intuitive, no extra UI

Cons: No keyboard support for power users

Option C: Drag to Pan

Drag viewport → Center on player
Tap tile → Move there

Pros: See more of the world, natural mobile UX

Cons: More complex to implement
